## Descrição

Projeto de uma api de produtos utilizando um framework derivado do Node.js chamado [Nest](https://github.com/nestjs/nest).
## Instalação


```bash
$ npm install
```


## Rodando a aplicação

```bash

# desenvolvimento
$ npm run start:dev

# produção
$ npm run start:prod

```

## Banco de dados

Neste projeto utilizei o Mysql como banco de dados, dentro do app temos um arquivo .env e um .env.example, onde neste último contém um exemplo de como a url deve ser configurada para poder rodar este projeto.

## Arquivo Insomnia

Dentro deste projeto temos uma pasta Insomnia onde contém as requests da nossa API, importe este arquivo no [Insomnia](https://insomnia.rest/).
